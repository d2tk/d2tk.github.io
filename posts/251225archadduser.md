### 안전하게 새로운 id 추가 및 비밀번호 설정

기존 유저명이 'a5'이고 새로운 유저 이름이 'mg'라고 가정을 해보겠습니다
아래의 명령을 통해 현재 로그인 되어 있는 'a5'로 sudo를 실행할 수 있는지 확인합니다

```
groups
```

출력에 `wheel` 또는 `sudo`가 포함되어 있으면 그대로 진행 가능합니다
만약 포함이 안되어 있다면 

```
sudo useradd -m -s /bin/bash mg
```
- `-m`은 홈 디렉토리를 생성
- `-s /bin/bash`는 기본 셸을 bash로 설정

제대로 되었다면 다음 명령에서 

```
ls /home
```

기존 id외에 mg가 나온다면 제대로 설정된 것입니다

이제 위의 내용까지 완료되었다면 패스워드 설정하고

```
sudo passwd mg
```

sudo 권한까지 부여합니다

```
sudo usermod -aG wheel mg
```

그룹확인 명령 
```
id mg
```

저는 다음과 같은 결과가 나왔습니다
uid=1001(mg) gid=1001(mg) groups=1001(mg),998(wheel)

이제 새로운 유저로 로그인하시려면 리부팅 하시거나 다음의 명령어로 전환하시면 됩니다

```
su -- mg  # 새로운 유저로 전환
```


### 데스크톱 환경 재시작 (선택 사항)

만약 SDDM을 사용 중이라면, SDDM을 재시작하여 새로운 유저가 로그인할 수 있도록 합니다

```
sudo systemctl restart sddm
```

### 주의사항

- 비밀번호를 설정할 때, 강력한 비밀번호를 사용하시기를 권장합니다
