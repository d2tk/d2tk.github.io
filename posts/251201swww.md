## swww 명령어로 이미지 파일 배경화면으로 만들기

```
swww img /home/a2/Desktop/wallpaper/iceberg_night.jpg
```

## 쉘 스크립트로 만들기

```
touch iceb_n.sh
```

 EDITOR로 만들 파일에 아래와 같은 내용으로 작성합니다

```
#!/bin/bash
swww img /home/a2/Desktop/wallpaper/iceberg_night.jpg
```

다 작성하고 저장한 뒤 chmod 명령어로 실행 권한을 부여합니다 

```
chmod +x ~/Desktop/iceb.n.sh
```
