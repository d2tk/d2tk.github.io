어떤 프로그램을 사용할지는 사실 취향의 문제이기도 합니다 

#### file manager
dolphin - KDE의 기본 파일 관리자로, 기능이 풍부하고 사용자 정의가 가능
thunar - dolphin보다 더 가볍고 심플
yazi - 더 가볍고 더 심플
lf - GUI 환경이 아닐 때 사용 주로 서버작업
[https://github.com/gokcehan/lf](a terminal file manager written in Go)

현재 thunar 사용 중

#### terminal emulator
kitty - GPU 가속을 지원하는 빠른 터미널 에뮬레이터로, 높은 성능과 다양한 기능을 제공
ghostty - 현대적인 디자인과 높은 성능을 제공하는 터미널 에뮬레이터

현재 ghostty 사용 중

#### web browser
firefox - Mozilla Firefox

그냥 linux에서는 이게 제일 만만합니다

#### media player
mpv - 강력한 미디어 플레이어

VLC media player 오래 쓰다가 mpv로 갈아 탔습니다

#### system monitoring
btop - 리소스 사용량을 실시간으로 모니터링할 수 있는 시스템 모니터링 도구

#### text editor
micro - 사용하기 쉬운 터미널 기반 텍스트 편집기로, 직관적인 인터페이스를 제공

#### IME 도구 (한국어 입력)
fcitx5 - 리눅스 환경에서 한글을 입력할 수 있도록 도와주는 도구
