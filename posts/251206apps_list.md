어떤 프로그램을 사용할지는 사실 취향의 문제이기도 합니다 

#### file manager

dolphin - KDE의 기본 파일 관리자로, 기능이 풍부하고 사용자 정의가 가능합니다
thunar - dolphin보다 더 가볍고 심플한 파일 관리자입니다
yazi - 가볍고 심플한 파일 관리자입니다
lf - GUI 환경이 아닐 때 주로 서버 작업에 사용합니다
[github 바로가기](https://github.com/gokcehan)

현재 Thunar를 주로 사용 중입니다

#### terminal emulator

kitty - GPU 가속을 지원하는 빠른 터미널 에뮬레이터로, 높은 성능과 다양한 기능을 제공합니다
ghostty - 현대적인 디자인과 높은 성능을 제공하는 터미널 에뮬레이터입니다

현재 Ghostty를 주로 사용 중입니다

#### web browser

firefox - Mozilla Firefox 웹브라우저 입니다
chrome - Antigravity의 브라우저 제어를 위해 설치한 웹 브라우저 입니다 

#### media player

mpv - 강력한 미디어 플레이어입니다

#### system monitoring

btop - 리소스 사용량을 실시간으로 모니터링할 수 있는 시스템 모니터링 도구입니다

#### 개발도구 및 텍스트 편집기

Antigravity - Google에서 만든 AI 도구입니다(VSCODE를 대신해서 사용중입니다)
zed - 빠르고 가볍고 Rust로 작성된 안정적인 편집기입니다
micro - 사용하기 쉬운 터미널 기반 텍스트 편집기로, 직관적인 인터페이스를 제공합니다

#### Disk 용량 체크

ncdu - 디스크 용량을 체크하는 도구입니다

#### 전자책 리더

Foliate - 전자책을 읽을 수 있는 리더입니다

#### image viewer

ristretto - 이미지를 볼 수 있는 뷰어입니다

#### IME 도구 (한국어 입력)

fcitx5 - 리눅스 환경에서 한글을 입력할 수 있도록 도와주는 도구 입니다

#### 음성녹음

audacity - 음성을 녹음할 수 있는 도구입니다
