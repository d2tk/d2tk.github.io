## 윈11에서 시스템 모니터링을 하는 방법

## resmon

cmd 환경에서 

```
resmon
```
명령문에서 알수 있다싶이 자원 모니터링 하는 명령입니다 
곧 설명할 Task manager에 비해 조금 구성이 다릅니다
(대신 반응은 더 빠를지도)
![[df.png]]

## ctrl + shift + esc
자주보던 Task manager 입니다
프로세스 및 성능 모니터링에 사용됩니다
