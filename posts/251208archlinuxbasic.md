## journalctl 명령어

시스템 로그를 확인하고 싶을 때 
창을 2개 띄워 놓고 한쪽에는 다음의 명령어를 실행합니다 

```
journalctl -f
```

## df 명령어

```
df -h
```

디스크 여유공간을 알아보고자 할 때 df(disk free)명령어를 통해 확인할 수 있습니다 
-h는 읽기 쉬운 형식으로 표시하라는 옵션입니다

## mkdir 명령어

폴더를 만들고자 할 때 다음의 명령어를 실행합니다 

mkdir 은 단일 폴더 생성 mkdir -p은 /a/b/c 와 같은 구조를 

```
mkdir -p [생성하려는 폴더명]
```


```
mkdir -p [생성하려는 폴더명]
```

## rm 명령어 

주의!!! : 폴더와 파일들이 삭제됩니다 
linux 시스템에서 가장 강력한 명령어 입니다 

- -f = 강제로 진행 - 확인 메시지 없음
- -r = 폴더 내의 모든 파일과 하위 폴더들까지 모두 삭제합니다 

파일 또는 폴더를 삭제하려고 할 때 다음의 명령어를 실행합니다

```
rm -rf [파일명] or [폴더명]
```

## pkill 프로세스 중단

```
pkill waybar
```

```
pkill dolphin
```

잠시 프로세스(애플리케이션)를 중단하고 다시 시작해야 할때 다음의 명령어를 자주 사용합니다

```
pkill dolphin && dolphin & 
```

명령문 구조 설명

```
# A && B &
# &&: 왼쪽 명령어(A)가 성공(종료 코드)하면 오른쪽 명령어(B) 실행
# & 뒤에 나온 & 는 백그라운드 실행 (터미널 즉시 반환)
```
