### Paru로 설치한 경우 업데이트는?
업데이트 명령은 다음과 같습니다
```
paru 패키지명
```

Google Antigravity를 업데이트 하는 경우
```
paru antigravity
```

### Paru 기본 명령어:
pacman의 명령과 거의 똑같다고 보면 됩니다 

#### 패키지 검색
```
paru -Ss 패키지명
```

#### 패키지 설치
```
paru -S 패키지명
```

#### 패키지 제거
```
paru -R 패키지명
```

보통은 의존성까지 제거해야 하기 때문에 아래를 더 많이 사용합니다
```
paru -Rs 패키지명
```

설정파일+의존성까지 제거하려면 다음의 명령어
```
paru -Rns 패키지명 # n은 설정파일까지 지우라는 옵션
```

#### 시스템 전체 업그레이드
```
paru -Syu
```

#### 공식저장소 패키지 확인
```
paru -Qn
```

#### 설치된 AUR 패키지 확인
```
paru -Qm
```

#### 공식 저장소 + 직접 설치한 것만
```
paru -Qne
```
- 공식 repo (`-n`)    
- 직접 설치 (`-e`)

#### 설치된 패키지 정보 확인
```
paru -Qi 패키지명
```

AUR 패키지 정보
```
paru -Si 패키지명
```
