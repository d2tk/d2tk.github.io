## pacman을 이용한 삭제

### 기본명령어
```
sudo pacman -R <패키지명>
```

### 기본명령어 및 추천명령어
 
```
sudo pacman -Rns <패키지명>
```

- '-n' 옵션은 설정 파일도 함께 제거
- '-s' 옵션은 의존성 패키지도 함께 제거

### 강제 명령어
주의: 강제 제거 명령은 시스템에 문제가 발생할 수 있습니다

```
sudo pacman -Rdd <패키지명>
```

- '-dd' 옵션은 의존성을 무시하고 강제로 제거합니다
- 의존성에 문제가 생긴경우 강제로 제거할 때 사용합니다
- 시스템에 문제가 발생할 수 있습니다
